STEP / 01 / 要件

Step 1 — html を作ってみよう

Step 1 — HTML बनाउने प्रयास गरौं

ChatGPT に依頼して、単語抽出ツールのHTMLを作る。

ChatGPT लाई अनुरोध गरेर शब्द निकाल्ने उपकरणको HTML बनाउनुहोस्
01
テキスト貼付エリア + 解析ボタン टेक्स्ट टाँस्ने ठाउँ + विश्लेषण बटन
02
結果表示
文字数 / 読了時間 / 段落数
परिणाम देखाउने — अक्षर संख्या / पढ्न लाग्ने समय / अनुच्छेद संख्या
03
英単語・カタカナ語を抽出してタグ表示
頻度順に並べる
अंग्रेजी शब्द र काताकाना शब्द निकालेर ट्याग देखाउने (आउने क्रममा)
04
HTML / CSS / JS の1ファイル完結
外部ライブラリ禁止
HTML / CSS / JS को १ फाइलमा सम्पन्न — बाह्य लाइब्रेरी निषेध
完了したら / पूरा भएपछि
AIに「このHTMLの設計書を書いて」と依頼して保存する。次のStepの土台になる。 AI लाई "यो HTML को डिजाइन डकुमेन्ट लेख्नुहोस्" भनेर अनुरोध गरेर सेभ गर्नुहोस्। अर्को Step को आधार बन्छ।
STEP / 02 / 要件

Step 2 — 拡張機能を作ってみよう

Step 2 — ब्राउजर एक्सटेन्सन बनाउने प्रयास गरौं

Step 1 のHTMLを Chrome 拡張機能にする。

Step 1 को HTML लाई Chrome एक्सटेन्सन बनाउनुहोस्
01
manifest.json + popup.html + content.js の3ファイル構成 manifest.json + popup.html + content.js — ३ फाइलको संरचना
02
「ページから取得」ボタンを追加
現在のタブの本文をテキストエリアに入れる
"पेजबाट लिने" बटन थप्नुहोस् — हालको ट्याबको मुख्य सामग्री टेक्स्ट क्षेत्रमा हाल्नुहोस्
03
Wikipedia / ニュース / 一般ブログで動作確認 Wikipedia / समाचार / सामान्य ब्लग मा परीक्षण गर्नुहोस्
04
chrome://extensions から読み込む
デベロッパーモードをオンにしてフォルダを選択
chrome://extensions बाट लोड गर्नुहोस् — डेभलपर मोड अन गरेर फोल्डर छनोट गर्नुहोस्
完了したら / पूरा भएपछि
AIに「この拡張機能の設計書を書いて」と依頼して保存する。次のStepの土台になる。 AI लाई "यो एक्सटेन्सनको डिजाइन डकुमेन्ट लेख्नुहोस्" भनेर अनुरोध गरेर सेभ गर्नुहोस्।
STEP / 03 / 要件

Step 3 — AI機能を追加してみよう

Step 3 — AI सुविधा थप्ने प्रयास गरौं

拡張機能から Anthropic Claude API を呼び出して3行要約を生成。

एक्सटेन्सनबाट Anthropic Claude API बोलाएर ३ हरफको सारांश बनाउनुहोस्
01
「3行で要約」ボタンを追加 "३ हरफमा सारांश" बटन थप्नुहोस्
02
Anthropic API を fetch で呼び出す
manifest.json に host_permissions と storage を追加
Anthropic API लाई fetch मार्फत बोलाउनुहोस् — manifest.json मा host_permissions र storage थप्नुहोस्
03
APIキー設定UI
chrome.storage.local に保存（コードには絶対書かない）
API कुञ्जी सेटिङ UI — chrome.storage.local मा सेभ (कोडमा कहिल्यै नलेख्नुहोस्)
完了したら / पूरा भएपछि
AIに「完成版ツールの設計書を書いて」と依頼して保存する。明日からの自分のリファレンスに。 AI लाई "पूर्ण उपकरणको डिजाइन डकुमेन्ट लेख्नुहोस्" भनेर अनुरोध गरेर सेभ गर्नुहोस्।
WARNING / APIキーの取り扱い

APIキーは
「鍵」かつ「請求書」

API कुञ्जी = "साँचो" र "बिल" दुवै हो

APIキーは「あなたの会社が払います」と証明する札。
これを持つ誰でも、あなたの口座でAIを呼び出せる。

API कुञ्जी = "तपाईंको कम्पनीले पैसा तिर्छ" भन्ने प्रमाण। यो भएकोले जोसुकैले तपाईंको खाताबाट AI चलाउन सक्छ।
❌ 絶対NG / पूर्ण निषेध

✅ 正しい扱い方 / सही तरिका
chrome.storage.local に保存chrome.storage.local मा सेभ गर्ने
設定UIから入力させるसेटिङ UI बाट इनपुट गराउने
1人1キーを守る१ व्यक्ति १ कुञ्जी राख्ने
流出に気づいたら即無効化लीक थाहा पाउनेबित्तिकै रद्द गर्ने
