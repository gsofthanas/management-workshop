AIを使う際の基本 / AI प्रयोग गर्दा आधारभूत कुरा

この5原則を使い倒します。コードはAIに書かせます。あなたは「何を作るか」と「困ったときの質問」だけ。

यी ५ सिद्धान्त राम्ररी प्रयोग गर्नुहोस्। कोड AI ले लेख्छ। तपाईंले गर्नु पर्ने भनेको "के बनाउने" र "अड्किँदा सोध्ने" मात्र हो।
  1. 整理する 何が欲しいかを自分の中で明確にしてから依頼する。 व्यवस्थित गर्नुहोस् — के चाहिएको आफूसँग स्पष्ट पारेर अनुरोध गर्नुहोस्।
  2. 具体的に伝える 曖昧を捨て、条件・制約・形式を明示する。 ठोस रूपमा भन्नुहोस् — अस्पष्टता हटाएर शर्त, बन्देज, ढाँचा स्पष्ट पार्नुहोस्।
  3. 構造化する ### 見出し でブロックを分けるとAIが理解しやすい。 संरचना दिनुहोस् — ### शीर्षक ले ब्लक छुट्याउँदा AI ले राम्ररी बुझ्छ।
  4. AIに先に質問させる プロンプトに自信がないなら「先に5つ質問して」と頼む。AIが要件を引き出してくれる。 AI लाई पहिले प्रश्न गर्न लगाउनुहोस् — प्रम्प्टमा विश्वास नभए "पहिले ५ वटा प्रश्न सोध" भन्नुहोस्। AI ले आवश्यकता निकालिदिन्छ।
  5. わからないことはAIに聞く 用語・概念・手順・エラー — 何でも遠慮なく。 नबुझेका कुरा AI लाई सोध्नुहोस् — शब्द, अवधारणा, तरिका, त्रुटि — जे पनि बेझिझक।

Step 1: HTMLを作ってみよう / HTML बनाउने प्रयास गरौं

テキストを貼り付けると文字数・読了時間・段落数・キーワードを表示するHTMLツールをAIに作らせます。

टेक्स्ट टाँस्दा अक्षर संख्या, पढ्न लाग्ने समय, अनुच्छेद संख्या, प्रमुख शब्दहरू देखाउने HTML उपकरण AI लाई बनाउन लगाउनुहोस्।

Step 2: 拡張機能を作ってみよう / ब्राउजर एक्सटेन्सन बनाउने प्रयास गरौं

Step 1 のHTMLを Chrome 拡張機能にして、任意のWebページから本文を取得できるようにします。

Step 1 को HTML लाई Chrome एक्सटेन्सन बनाएर, जुनसुकै वेब पेजबाट मुख्य सामग्री लिन सकिने बनाउनुहोस्।

AIへの依頼 / AI लाई अनुरोध

Step 1 の設計書と一緒に渡す / Step 1 को डिजाइन डकुमेन्टसँगै दिनुहोस् このHTMLツールを Chrome 拡張機能（Manifest V3）にしたい。 - 「📥 このページから取得」ボタンを popup に追加 - ボタンを押すと現在のタブの本文を抽出してテキストエリアに入れる - Wikipedia / ニュース / 一般記事 で動くようにする manifest.json / popup.html / popup.js / content.js を作ってください。 迷う点があれば先に質問してください。

拡張機能を Chrome に読み込む手順 / Chrome मा एक्सटेन्सन लोड गर्ने तरिका

これだけは AI ではなく、自分の手で操作します。

यो मात्रै AI ले होइन, आफैले गर्नुपर्छ।
  1. chrome://extensions をアドレスバーに入力chrome://extensions लाई एड्रेस बारमा टाइप गर्नुहोस्
  2. 右上の「デベロッパーモード」をオンमाथि दायाँको "डेभलपर मोड" अन गर्नुहोस्
  3. 「パッケージ化されていない拡張機能を読み込む」をクリック"Load unpacked" क्लिक गर्नुहोस्
  4. AIが作ったファイルを置いたフォルダを選択AI ले बनाएको फाइलहरू भएको फोल्डर छनोट गर्नुहोस्
  5. ツールバーの拡張アイコンから起動टुलबारको एक्सटेन्सन आइकनबाट चलाउनुहोस्

完成チェック / पूर्णताको जाँच

Wikipedia や ニュース記事を開いて、拡張アイコン → 「ページから取得」 で本文がテキストエリアに入れば成功。

Wikipedia वा समाचार लेख खोलेर, एक्सटेन्सन आइकन → "पेजबाट लिने" मार्फत मुख्य सामग्री टेक्स्ट क्षेत्रमा आयो भने सफल।

完了したら / पूरा भएपछि

AIに 「この拡張機能の設計書をMarkdownで書いて」 と依頼。Step 3 でAIに渡す土台になる。

AI लाई "यो एक्सटेन्सनको डिजाइन डकुमेन्ट Markdown मा लेख्नुहोस्" भनेर अनुरोध गर्नुहोस्। Step 3 मा AI लाई दिने आधार बन्छ।

Step 3: AI機能を追加してみよう / AI सुविधा थप्ने प्रयास गरौं

拡張機能から Anthropic Claude API を呼び出して、本文を3行に要約します。

एक्सटेन्सनबाट Anthropic Claude API बोलाएर, मुख्य सामग्रीलाई ३ हरफमा सारांश बनाउनुहोस्।

⚠️ APIキーの大原則 / API कुञ्जीको मुख्य नियम

APIキーは chrome.storage.local にだけ保存。ソースコードに絶対書かない。Gitに上げない。Slack・メール・画面共有も全部NG。流出すると一晩で数百万円の被害になる事例あり。

API कुञ्जी chrome.storage.local मा मात्र सेभ गर्नुहोस्। सोर्स कोडमा कहिल्यै नलेख्नुहोस्। Git मा नहाल्नुहोस्। Slack/इमेल/स्क्रिन सेयर पनि सबै निषेध। लीक भयो भने एक रातमै लाखौंको क्षति हुने उदाहरण छन्।

AIへの依頼 / AI लाई अनुरोध

Step 2 の設計書と一緒に渡す / Step 2 को डिजाइन डकुमेन्टसँगै दिनुहोस् この拡張機能に「✨ 3行で要約」ボタンを追加したい。 - ボタンを押すとテキストエリアの内容を Anthropic Claude API に送る - 結果を3行（結論 → 理由 → 使い所）で表示する - APIキーは設定UIから入力させて chrome.storage.local に保存 - ソースコードに絶対書かない - manifest.json に必要な権限を追加 迷う点があれば先に質問してください。

システムプロンプトのヒント / सिस्टम प्रम्प्टको सङ्केत

「3行要約のシステムプロンプトを作って」と AI に頼むときは、こんな要素を含めるよう指定すると精度が上がります。

"३ हरफको सारांशको सिस्टम प्रम्प्ट बनाउनुहोस्" भनेर AI लाई अनुरोध गर्दा, यी तत्वहरू समावेश गर्न भन्दा गुणस्तर बढ्छ।
  • 役割 — 「あなたは記事を3行で要約するエキスパートです」भूमिका — "तपाईं ३ हरफमा सारांश गर्ने विशेषज्ञ हुनुहुन्छ"
  • 出力の型 — 結論 / 理由 / 使い所 の順आउटपुट ढाँचा — निष्कर्ष / कारण / प्रयोगको ठाउँ
  • 文字数制限 — 各行40文字以内अक्षर सीमा — हरेक हरफ ४० अक्षर भित्र
  • 出力フォーマット — 番号や記号なし、改行のみआउटपुट फर्म्याट — नम्बर वा चिन्ह नहाल्ने, हरफ ब्रेक मात्र

完成チェック / पूर्णताको जाँच

ニュース記事を開いて「ページから取得」→「3行で要約」で、3行の要約が表示されたら成功。

समाचार लेख खोलेर "पेजबाट लिने" → "३ हरफमा सारांश" मार्फत ३ हरफको सारांश देखियो भने सफल।

完了したら / पूरा भएपछि

AIに 「完成版ツールの設計書をMarkdownで書いて」 と依頼。明日からの自分のリファレンスになる。

AI लाई "पूर्ण उपकरणको डिजाइन डकुमेन्ट Markdown मा लेख्नुहोस्" भनेर अनुरोध गर्नुहोस्। भोलिदेखि आफ्नो सन्दर्भ बन्छ।

ボーナス課題 / बोनस चुनौती

最低限が動いたらこれらを試そう。全部 AI に頼めば実装できる。

न्यूनतम चलेपछि यी चुनौती गर्नुहोस्। सबै AI लाई अनुरोध गरेर बनाउन सकिन्छ।

モードを増やす / मोड थप्ने

システムプロンプトを変えるだけで別のモードが作れる。AIに「○○モードのシステムプロンプトを書いて」と頼む。

सिस्टम प्रम्प्ट परिवर्तन गर्दा अर्को मोड बन्छ। AI लाई "○○ मोडको सिस्टम प्रम्प्ट लेख्नुहोस्" भनेर अनुरोध गर्नुहोस्।
  • やさしい日本語モード — 専門用語を中学生にもわかる言葉にसजिलो जापानी मोड — विशेषज्ञ शब्दलाई कक्षा ८ का विद्यार्थीले बुझ्ने भाषामा
  • 翻訳モード — 英語ページを自然な日本語にअनुवाद मोड — अंग्रेजी पेजलाई स्वाभाविक जापानीमा
  • TODO抽出モード — 議事録からアクションアイテムだけ抽出TODO निकाल्ने मोड — मिनेट्सबाट कार्य आइटम मात्र निकाल्ने

機能を追加する / सुविधा थप्ने

AIに「○○機能を追加して」と個別に頼めば、既存コードに足してくれる。

AI लाई "○○ सुविधा थप्नुहोस्" भनेर अलग्गै अनुरोध गर्दा, अहिलेको कोडमा थपिदिन्छ।
  • 履歴機能 — 過去の要約を chrome.storage.local に保存इतिहास सुविधा — विगतका सारांश chrome.storage.local मा सेभ
  • コピーボタン — 要約結果をクリップボードにコピーकपी बटन — सारांश परिणाम क्लिपबोर्डमा कपी
  • 読み上げ機能 — 音声で読み上げ（Web Speech API）पढ्ने सुविधा — आवाजमा पढ्ने (Web Speech API)
  • ダークモード — 目に優しい配色डार्क मोड — आँखालाई कोमल रङ

トラブルシューティング / समस्या समाधान

用語や概念がわからない / शब्द वा अवधारणा नबुझ्दा

その場でAIに聞いてください。これがこの研修の重要な学びです。

तत्कालै AI लाई सोध्नुहोस्। यो नै यो प्रशिक्षणको महत्त्वपूर्ण सिकाइ हो।
質問例 / प्रश्न नमुना manifest.json って何？初心者にもわかるように 5 行で説明して。 content script と popup script の違いを、子供に説明するように教えて。 CORS って何？なぜ起きるの？

AIが出したコードがエラーになる / AI को कोडमा त्रुटि आउँदा

  • エラーメッセージをそのままコピーして AI に貼るत्रुटि सन्देश सीधै कपी गरेर AI लाई टाँस्नुहोस्
  • 「このエラーが出ました。原因と修正方法を教えて」と聞く"यो त्रुटि आयो। कारण र समाधान भन्नुहोस्" भनेर सोध्नुहोस्
  • 何度かやり取りすればだいたい直るकेही पटक पठाएपछि प्रायः ठीक हुन्छ

拡張機能が読み込めない / एक्सटेन्सन लोड नहुँदा

  • chrome://extensions でエラーメッセージを確認chrome://extensions मा त्रुटि सन्देश हेर्नुहोस्
  • そのエラーをAIに貼って「直して」と頼むत्यो त्रुटि AI लाई टाँसेर "ठीक गर्नुहोस्" भन्नुहोस्
  • ファイル名が manifest.json になっているかफाइलको नाम manifest.json छ कि छैन जाँच्नुहोस्

「ページから取得」が動かない / "पेजबाट लिने" काम नगर्दा

  • そのサイトのURLを AI に教えて「このサイトから本文を取れるようにして」と頼むसाइटको URL AI लाई दिएर "यो साइटबाट मुख्य सामग्री लिने बनाउनुहोस्" भन्नुहोस्
  • セレクタ調整は AI の得意分野सेलेक्टर मिलाउने कुरा AI को दक्ष क्षेत्र हो

APIキーエラー / API कुञ्जी त्रुटि

  • キーが正しく保存されているか設定UIで確認कुञ्जी ठीक सेभ भएको छ कि छैन सेटिङ UI मा हेर्नुहोस्
  • キーの先頭が sk-ant-api03- で始まっているかकुञ्जीको सुरु sk-ant-api03- ले सुरु हुन्छ कि छैन
  • 余分なスペースや改行が混じっていないかअतिरिक्त स्पेस वा हरफ ब्रेक मिसिएको छ कि छैन
