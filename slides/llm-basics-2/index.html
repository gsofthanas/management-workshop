STEP / 01 / 要件

Step 1 — html を作ってみよう

ChatGPT に依頼して、単語抽出ツールのHTMLを作る。

01
テキスト貼付エリア + 解析ボタン
02
結果表示
文字数 / 読了時間 / 段落数
03
英単語・カタカナ語を抽出してタグ表示
頻度順に並べる
04
HTML / CSS / JS の1ファイル完結
外部ライブラリ禁止
完了したら
AIに「このHTMLの設計書を書いて」と依頼して保存する。次のStepの土台になる。
STEP / 02 / 要件

Step 2 — 拡張機能を作ってみよう

Step 1 のHTMLを Chrome 拡張機能にする。

01
manifest.json + popup.html + content.js の3ファイル構成
02
「ページから取得」ボタンを追加
現在のタブの本文をテキストエリアに入れる
03
Wikipedia / ニュース / 一般ブログで動作確認
04
chrome://extensions から読み込む
デベロッパーモードをオンにしてフォルダを選択
完了したら
AIに「この拡張機能の設計書を書いて」と依頼して保存する。次のStepの土台になる。
STEP / 03 / 要件

Step 3 — AI機能を追加してみよう

拡張機能から Anthropic Claude API を呼び出して3行要約を生成。

01
「3行で要約」ボタンを追加
02
Anthropic API を fetch で呼び出す
manifest.json に host_permissions と storage を追加
03
APIキー設定UI
chrome.storage.local に保存（コードには絶対書かない）
完了したら
AIに「完成版ツールの設計書を書いて」と依頼して保存する。明日からの自分のリファレンスに。
WARNING / APIキーの取り扱い

APIキーは
「鍵」かつ「請求書」

APIキーは「あなたの会社が払います」と証明する札。
これを持つ誰でも、あなたの口座でAIを呼び出せる。

❌ 絶対NG
ソースコードに直書き
Git にコミット
Slack・メールで共有
公開ページのJSに埋め込む
✅ 正しい扱い方
chrome.storage.local に保存
設定UIから入力させる
1人1キーを守る
流出に気づいたら即無効化

流出すると一晩で数百万円の被害になる事例あり
