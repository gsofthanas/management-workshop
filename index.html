Tech Grid Theme

css/tech-grid.css を読み込み、.slide に .tech-grid を追加すると有効化される opt-in テーマ。
ドットグリッド背景・L字コーナーマーク・モノスペースタグ・ターミナル風コードブロックなどを提供。

使い方

<link rel="stylesheet" href="css/pres.css">
<link rel="stylesheet" href="css/tech-grid.css">

<div class="slide tech-grid">
  <span class="tg-corner tr"></span>
  <span class="tg-corner bl"></span>
  <span class="tg-tag">SECTION / 01</span>
  <h2 class="tg-mega bad">守る</h2>
</div>

提供コンポーネント

.slide.tech-grid

ドットグリッド背景。スライドに付与するだけで Tech Grid 化。

.tg-corner.tl/.tr/.bl/.br

L字コーナーマーク。4位置 + bad/accent/ok/gray の色バリアント。

.tg-tag

[ MONO LABEL ] モノスペースのブラケットラベル。

.tg-mega

巨大セクションタイトル (200px超)。セクション扉用。

.tg-agenda / .tg-agenda-card

3カラム目次カード。各カードに小L字角 + セクション色。

.tg-code-block

ターミナル風コードブロック。実習スライド用 (信号機 + シンタックスハイライト)。

.tg-rule-list / .tg-rule-item

番号付きまとめリスト + 破線区切り + // コメント形式サブ。

.tg-meta

下部メタ情報帯。FILE名・スライド数などを左右に配置。

.tg-divider-text / .tg-rule

横線付き区切りテキストとルールバー。
